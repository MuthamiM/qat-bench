Transformer vs QAT Blocks

PyTorch FBGEMM Limitations

Our 15M parameter Causal Language Model leverages a custom QATBlock. Native CPU quantized execution explicitly lacks support for Batch Matrix Multiplication (bmm) and LayerNorms in INT8 space.

By wrapping the Attention matrix and normalization steps with DeQuantStub(), we seamlessly hop the tensors into FP32 specifically for calculation, bridging the hardware divide.
